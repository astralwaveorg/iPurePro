# iPure·Pro

一款基于 Hamster 输入法的皮肤配置，采用 Jsonnet 模板语言生成，支持亮色和暗色两种主题。

## 简介

StellarX 是一个输入法皮肤项目，通过 Jsonnet 配置文件生成键盘布局，主要特点包括：

- 使用 SF Symbols 图标系统
- 支持中英文输入
- 提供滑动和长按功能
- 适配 iPhone 和 iPad 设备

## 技术实现

### 配置结构

项目使用 Jsonnet 作为配置语言，主要文件结构：

```
wechat/
├── config.yaml              # 皮肤配置元数据
├── jsonnet/                 # Jsonnet 源文件
│   ├── main.jsonnet        # 主入口文件
│   ├── keyboard/           # 键盘布局定义
│   └── lib/               # 库文件（共享配置）
└── light/ / dark/          # 生成的配置文件
```

### 键盘类型

支持以下键盘类型：

- **拼音键盘 (pinyin_26)**: 中文拼音输入
- **字母键盘 (alphabetic_26)**: 英文输入
- **数字键盘 (numeric_9)**: 数字输入
- **符号键盘 (symbolic)**: 符号输入

### 功能特性

#### 滑动功能

通过上下滑动按键可以快速输入符号或执行快捷操作：

- **上滑**：数字、符号、快捷操作（全选、剪切、复制、粘贴等）
- **下滑**：特殊符号、快捷操作（撤销、重做等）

#### 长按功能

长按按键显示候选字符或快捷操作。

#### 智能回车

根据输入框类型自动显示不同的回车文字：
- 回车、搜索、发送、前往、完成

#### Shift 键

按下 Shift 时显示绿色背景，指示当前输入状态。

## 键盘布局

### 主键盘布局

主键盘采用 26 键布局，底部功能按键：

- **符**：切换到符号选择键盘
- **符号**：中英文切换
- **数字**：切换到数字键盘
- **空格**：输入空格
- **逗号**：输入标点符号
- **回车**：确认输入

## 安装方法

1. 下载皮肤文件
2. 将文件放入 Hamster 输入法的皮肤目录
3. 在输入法设置中选择 StellarX 皮肤

## 技术规格

- **配置格式**：Jsonnet + YAML
- **图标系统**：SF Symbols (iOS 15+)
- **支持版本**：Hamster 输入法 v3+
- **主题模式**：亮色主题 / 暗色主题

## 配置参数

### 字体大小
- 中文按键：18pt
- 英文按键：19pt
- 滑动符号：12pt
- 工具栏图标：18pt

### 按键位置
- 字母按键垂直偏移：0.55
- 功能键偏移：0.45

### 颜色配置
- 主色调：#23C891
- 亮色主题：#000000（前景），#FFFFFF（背景）
- 暗色主题：#FFFFFF（前景），#1C1C1E（背景）

## 构建方法

在皮肤目录执行：

```bash
jsonnet -S -m . jsonnet/main.jsonnet
```

该命令会读取 `jsonnet/main.jsonnet` 作为入口，生成所有键盘配置文件，输出到对应的 `light/` 和 `dark/` 目录。
